# Инструкция по запуску приложения офлайн

## Как получить версию для Windows без интернета

### Шаг 1: Получить код проекта
1. В редакторе poehali.dev нажмите **Скачать → Скачать билд**
2. Вы получите ZIP-архив со всеми файлами

### Шаг 2: Распаковать и запустить
1. Распакуйте ZIP-архив в любую папку на компьютере
2. Найдите файл `index.html` в корне папки
3. Дважды кликните по `index.html` - игра откроется в браузере

**Важно:** Все файлы остаются на вашем компьютере, интернет не нужен!

---

## Альтернатива: Создать .exe файл (требует технических знаний)

Если нужен настоящий .exe файл:

### Вариант 1: Tauri (легкий, ~3-5 МБ)
1. Подключите GitHub через **Скачать → Подключить GitHub**
2. Склонируйте репозиторий на компьютер
3. Установите Rust: https://www.rust-lang.org/
4. Следуйте инструкции Tauri: https://tauri.app/

### Вариант 2: Electron (полный, ~100 МБ)
1. Подключите GitHub через **Скачать → Подключить GitHub**
2. Склонируйте репозиторий на компьютер
3. Установите Node.js: https://nodejs.org/
4. Используйте Electron Builder для создания .exe

---

## Быстрый способ (рекомендуется)
Просто запустите `index.html` из скачанного билда - этого достаточно для игры без интернета!
